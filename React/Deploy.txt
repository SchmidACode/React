1. npx create-react-app appname;

2. Залить репозиторий на GitHub;

3. На странице репозитория зайти в 
	Settings -> Pages -> Выбрать ветку 'master' и нажать 'Save',
	после чего, через короткий промежуток времени вверху появится 
	ссылка на приложение.

4. Полученную ссылку добавить в 'package.json':
	{
		"name": "reactintro",
		"version": "0.1.0",
!!!		"homepage": "https://okovtun.github.io/ReactIntro",
		..........
		..........

5. Так же, в 'package.json', в секцию 'scripts' добавить параметры 'predeploy' и 'deploy':
		  "scripts": {
			"predeploy": "npm run build",
			"deploy": "gh-pages -d build",
			........
			........

6. Установить в проект GitHub-Pages:
	npm install --save gh-pages

7. Выполнить развертывание:
	npm run deploy

8. В настройках репозитория на GitHub:
	Settings -> Pages -> поменять ветку с 'master' на 'gh-pages' и нажать 'Save';

9. Примерно через 2-3 минуты страница будет развернута на хостинге GitHub;
